# Mother's Day Gift Website

This is a mobile-first single-page Mother's Day website created for **Josephine Yaba Dzikunu (Mama Joe)**. It uses a warm gold-and-orange visual style, soft animation, a heartfelt letter, a responsive gallery, a photo-led memories timeline, a surprise reveal, and background music controls.

## Files

- `index.html` contains the page structure, personalized content, and section markup.
- `style.css` contains the visual system, layout, responsive styling, and motion behavior.
- `script.js` contains the typing effect, scroll reveals, accessible gallery lightbox, surprise interaction, and music control.

## Assets

- Main gallery images live in `assets/images/optimized/`.
- Timeline photos live in `assets/images/timeline/`.
- Background music is loaded from `assets/audio/sweet-mother.mp3`.

## How to Use

Open `index.html` in a browser to preview the site locally.
