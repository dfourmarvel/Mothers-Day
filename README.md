# Mother's Day Gift Website

This is a mobile-first single-page Mother's Day website created for **Josephine Yaba Dzikunu (Mama Joe)**. It uses a warm gold-and-orange visual style, soft animation, a heartfelt letter, a responsive gallery, a memories timeline, a surprise reveal, and background music controls.

## Files

- `index.html` contains the full page structure and placeholder content.
- `style.css` contains the visual design, layout, gradients, and animations.
- `script.js` contains the typing effect, scroll reveals, gallery lightbox, surprise interaction, and music control.

## Replace Later

- Swap the placeholder gallery image URLs in `index.html` with your real photos.
- Add your audio file at `assets/mama-joe-song.mp3`, or update the audio path in `index.html`.

## How to Use

Open `index.html` in a browser to preview the site locally.
